# GitHub Pages Setup

1. Create a GitHub repo for this project, or push these `docs/` files to an existing repo.
2. Push the repo to GitHub.
3. On GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, set:
   - `Source`: `Deploy from a branch`
   - `Branch`: your main branch
   - `Folder`: `/docs`
5. Save.
6. GitHub will publish a site at:
   - `https://YOUR-USERNAME.github.io/YOUR-REPO/`

Useful URLs after publish:
- Privacy policy: `https://YOUR-USERNAME.github.io/YOUR-REPO/privacy-policy.html`
- Support: `https://YOUR-USERNAME.github.io/YOUR-REPO/support.html`

Use those links in App Store Connect for the Privacy Policy URL and Support URL.
